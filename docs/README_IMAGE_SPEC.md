# README画像 仕様データ（現行版）

## Version

v0.4.0

## Main Features

現在搭載されている全機能。バッジ画像には省略せず全項目を描画する。

1. ファイル検索 — 設定したフォルダ内のファイルをインクリメンタル検索。使用頻度に応じて賢くランク付け（frecency）。
2. 計算機能 — 「1000 + 200」のように入力するとすぐに計算結果を表示。「(1+3)/3」のように括弧を使った優先順位付きの計算にも対応。
3. クリップボード履歴 — 「/cb」でテキストや画像の履歴を一覧表示。コピー＆復元も簡単。呼び出しキーワードは設定で変更可能。
4. OCR（画像→テキスト） — 画像をCtrl+Vで貼り付けると、Windows OCRで文字を抽出。日本語・英語の混在にも対応。
5. Web検索 — Googleを直接検索。
6. システムコマンド — 「/shutdown」「/restart」「/sleep」でシャットダウン・再起動・スリープを確認付きで実行。キーワードはコマンドごとにカスタマイズ可能。
7. カスタマイズ可能なホットキー — デフォルトは Alt + Space。設定画面から変更可能。
8. システムトレイ常駐 — バックグラウンドで静かに動作。

補足：「/」を入力すると、利用可能なプレフィックスコマンド（上記3・6）の候補が使用頻度順に一覧表示される。検索画面モックアップ横の吹き出しで軽く触れる程度でよい。

## Shortcuts

| キー | 説明 |
| --- | --- |
| `Alt+Space` | ランチャーを開く／閉じる |
| `↑↓` | 結果を選択 |
| `Enter` | 起動／実行 |
| `Esc` | 閉じる |
| `Ctrl+S` | 設定画面を開く／閉じる |
| `/cb` | クリップボード履歴を開く |
| `/shutdown` | シャットダウン（確認あり） |
| `/restart` | 再起動（確認あり） |
| `/sleep` | スリープ（確認あり） |
| `Ctrl+V`（画像） | 画像を貼り付けてOCR |

## Screenshots

中央下に並べる4枚のモックアップ対象（この4画面以外は描かない）。

1. ファイル検索 — 検索ボックス＋アイコン付きファイル一覧
2. 設定画面 — 左にカテゴリナビ（全般／ファイル検索／数式計算／システムコマンド／Web検索／クリップボード等）、右に選択中カテゴリの設定項目
3. クリップボード履歴 — 「/cb」で呼び出す左右2カラムパネル（左：履歴一覧、右：選択中エントリのプレビュー）
4. OCR — 左に貼り付けた画像のプレビュー、右にOCRで抽出したテキストと「コピーして閉じる」「閉じる」ボタン
