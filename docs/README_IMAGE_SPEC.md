# README画像 仕様データ（現行版）

## Version

v0.6.0

## Main Features

現在搭載されている全機能。バッジ画像には省略せず全項目を描画する。

1. ファイル検索 — 設定したフォルダ内のファイルをインクリメンタル検索。使用頻度に応じて賢くランク付け（frecency）。
2. 計算機能 — 「1000 + 200」のように入力するとすぐに計算結果を表示。「(1+3)/3」のように括弧を使った優先順位付きの計算にも対応。
3. クリップボード履歴 — 「/cb」でテキストや画像の履歴を一覧表示。コピー＆復元も簡単。呼び出しキーワード・最大保持件数（1〜200件）は設定で変更可能。
4. 最近使ったファイル一覧 — 「/recent」でWindows・Office（Excel/Word/PowerPoint）に加え、OneDrive（個人・OneDrive for Business・SharePointの共有ライブラリ/ショートカットを含む）上のファイルも含めた最近使った履歴を一覧表示。呼び出しキーワード・保持期間（デフォルト180日）・最大表示件数（デフォルト50件）は設定で変更可能。
5. 格納フォルダを開く — 検索結果でShift+Enterを押すと、選択中のファイルの格納フォルダをエクスプローラーで開けます。ショートカット（.lnk）はリンク先の実ファイルのフォルダを開きます。
6. OCR（画像→テキスト） — 画像をCtrl+Vで貼り付けると、Windows OCRで文字を抽出。日本語・英語の混在にも対応。
7. Web検索 — Googleを直接検索。
8. システムコマンド — 「/shutdown」「/restart」「/sleep」でシャットダウン・再起動・スリープを確認付きで実行。キーワードはコマンドごとにカスタマイズ可能。
9. カスタマイズ可能なホットキー — デフォルトは Alt + Space。設定画面から変更可能。
10. システムトレイ常駐 — バックグラウンドで静かに動作。

補足：「/」を入力すると、利用可能なプレフィックスコマンド（上記3・4・8）の候補が使用頻度順に一覧表示される。検索画面モックアップ横の吹き出しで軽く触れる程度でよい。

## Shortcuts

| キー | 説明 |
| --- | --- |
| `Alt+Space` | ランチャーを開く／閉じる |
| `↑↓` | 結果を選択 |
| `Enter` | 起動／実行 |
| `Shift+Enter` | 選択中のファイルの格納フォルダを開く |
| `Esc` | 閉じる |
| `Ctrl+S` | 設定画面を開く／閉じる |
| `/cb` | クリップボード履歴を開く |
| `/recent` | 最近使ったファイル一覧を開く |
| `/shutdown` | シャットダウン（確認あり） |
| `/restart` | 再起動（確認あり） |
| `/sleep` | スリープ（確認あり） |
| `Ctrl+V`（画像） | 画像を貼り付けてOCR |
