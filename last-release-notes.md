## What's Changed

- Added Restart option to the tray menu
- Added OCR (image-to-text) support via Windows OCR API, with mixed Japanese/English recognition
- Paste an image (Ctrl+V) into the search box to extract text
- Improved word spacing accuracy for CJK/ASCII mixed text
- Lines sorted by bounding box Y-coordinate for correct reading order
- Improved OCR preview UX: focus returns to the search box on close, and the window fades out smoothly on copy-and-close
- Version number now displayed automatically in settings footer
- Fixed folder search results to open in Explorer reliably
- Added security note to README regarding unencrypted clipboard history
